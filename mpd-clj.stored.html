list-playlist

(list-playlist name mpd-server)
lists songs in the playlist

list-playlist-info

(list-playlist-info name mpd-server)
lists songs in the playlist. with metadata

list-playlists

(list-playlists mpd-server)
show saved playlists

load-playlist

(load-playlist name mpd-server)(load-playlist name range mpd-server)
loads the playlist into the current queue. RANGE is optional

playlist-add

(playlist-add name uri mpd-server)
adds URI to the playlist NAME.m3u. NAME.m3u will be created if does not exist

playlist-clear

(playlist-clear name mpd-server)
clears the playlist NAME.m3u

playlist-delete

(playlist-delete name songpos mpd-server)
deletes SONGPOS from the playlist NAME.m3u

playlist-move

(playlist-move name songid songpos mpd-server)
moves SONGID in the playlist NAME.m3u to SONGPOS

playlist-rename

(playlist-rename name newname mpd-server)
renames the playlist NAME.m3u to NEWNAME.m3u

playlist-rm

(playlist-rm name mpd-server)
removes the playlist NAME.m3u from the playlist directory

playlist-save

(playlist-save name mpd-server)
saves the current playlist to NAME.m3u in the playlist directory
